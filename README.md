
# RideOn Flask App (SQLite)
This is a minimal Flask application for the RideOn Bike Rental project.
## How to run
1. (Optional) create a virtualenv: python -m venv venv && source venv/bin/activate
2. Install requirements: pip install -r requirements.txt
3. Run the app: python app.py
4. Open http://127.0.0.1:5000/
## Files created
- app.py : main Flask application
- rideon_bike_rental.db : pre-populated SQLite database (10 rows per table)
- templates/* : HTML templates for pages
